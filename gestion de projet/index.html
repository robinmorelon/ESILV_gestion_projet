🚙 Régimes d'usage (regpar)

  • Mixte : Tarification visiteur standard, préférentielle pour les résidents.
  • Rotatif : Stationnement de courte durée pour favoriser l'activité commerciale.
  • Livraison : Espaces logistiques professionnels.
  • PMR / GIG-GIC : Emplacements réservés à l'accessibilité.

💳 Régime Tarifaire (regpri)

  • Payant : Assujetti au Secteur Tarifaire Visiteur (STV) en vigueur.
  • Gratuit : Zones non réglementées.

📐 Configuration Spatiale (typsta)

  • Longitudinal : Stationnement aligné sur la voirie (Créneau).
  • Bataille / Épi : Stationnement perpendiculaire ou angulaire.
